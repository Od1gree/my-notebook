从外部导入数据

导入CSV文件

  1. 使用excel类软件填入数据
  2. 注意各种数据类型和数据库内容相匹配
  3. 注意文本编码输出utf8格式
  4. 使用以下code进行填入,其中第一行为表头所以ignore第一行.
LOAD DATA LOCAL INFILE '/home/erik/Documents/out10-2.csv'
INTO TABLE out10
FIELDS TERMINATED BY ','
IGNORE 1 LINES
;
